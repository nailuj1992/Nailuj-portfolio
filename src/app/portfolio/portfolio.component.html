Visual environment for learning propositional logic
The general objective of the project consists mainly of designing and building software with a highly visual interface, based on a playful metaphor, that allows the student to learn concepts related to propositional logic in an easier and more motivating way, in order to provide a suitable space for experimentation and conjecture.

Basic music player
This is a basic music player, which is developed in Java. This player supports MP3 audio format files and offers basic controls such as volume adjustment and track progression. While maintaining a minimalist approach, this music player serves as an skeleton that can be expanded upon to incorporate more advanced features based on user preferences.
